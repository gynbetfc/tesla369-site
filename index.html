📱 Por que o Termux?

O Termux é um aplicativo gratuito que roda um terminal Linux dentro do seu Android. É ele que executa o bot diretamente no seu celular, usando seu 4G/WiFi. Isso significa que a IQ Option vê o IP do seu celular, evitando bloqueios que acontecem quando muitos usuários usam o mesmo servidor.

📲 Como Instalar

1

Instale o Termux

É um app gratuito da Play Store (sem anúncios, seguro). Ele cria um ambiente Linux no seu celular para rodar o bot.

▶️ Baixar Termux
2

Abra o Termux e cole o comando

Isso vai instalar tudo automaticamente. É só colar e esperar.

bash <(curl -4 -s https://raw.githubusercontent.com/gynbetfc/v-sensitivo-bot/main/install.sh)
3

Crie o atalho (1 vez só)

Depois de instalar, cole este comando para criar um atalho:

echo "alias tesla='bash <(curl -4 -s https://raw.githubusercontent.com/gynbetfc/v-sensitivo-bot/main/install.sh)'" >> ~/.bashrc && source ~/.bashrc

✅ Pronto! Agora é só digitar tesla no Termux pra iniciar!
